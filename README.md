# EyeTracker

## Обзор

Система отслеживания взгляда через веб-камеру для настольных приложений. Принцип работы:

1. **Детекция лица** — MediaPipe Face Mesh находит 468 лицевых ландмарков
2. **Извлечение патчей глаз** — из видеокадра вырезаются области левого и правого глаза
3. **Извлечение признаков** — каждый глаз масштабируется до 10x6 пикселей, переводится в градации серого и выравнивается по гистограмме (вектор 120D)
4. **Калибровка** — пользователь кликает по точкам на экране, глядя на них; система обучает маппинг: признаки глаз -> координаты на экране
5. **Предсказание взгляда** — взвешенная гребневая регрессия в реальном времени предсказывает координаты взгляда по признакам глаз

## Требования

- Python 3.12+
- Веб-камера
- macOS / Windows

## Сборка

Приложение собирается в нативный исполняемый файл через PyInstaller:

```bash
make install   # установка зависимостей (включая PyInstaller)
make build     # сборка приложения
```

Результат:
- **macOS** — `dist/EyeTracker.app` (подписывается ad-hoc с entitlements для доступа к камере)
- **Windows** — `dist/EyeTracker.exe` (single-file)

На macOS при первом запуске система запросит разрешение на доступ к камере.

## Запуск

### Собранное приложение

- **macOS**: открыть `dist/EyeTracker.app` или `open dist/EyeTracker.app`
- **Windows**: запустить `dist\EyeTracker.exe`

### Из исходников

```bash
make run
```

Или напрямую:

```bash
poetry run eyetracker
```

### Процесс калибровки

Откроется полноэкранное окно калибровки:
1. Кликните по каждой из 9 красных точек 5 раз, глядя на них
2. Смотрите на зелёную точку 5 секунд (измерение точности)
3. Автоматический переход в режим отслеживания взгляда (красная точка следует за глазами)

### Превью видео

В левом верхнем углу отображается превью с веб-камеры и наложением ландмарков лица. Нажмите **V** чтобы показать/скрыть превью (работает на любом этапе).

### Режим динамической тренировки

В режиме отслеживания взгляда нажмите **T** для включения режима тренировки. В этом режиме кликайте по экрану, глядя на точку клика — модель будет дообучаться на лету.

### Параметры командной строки

```bash
poetry run eyetracker --help

Параметры:
  --camera INDEX       Индекс устройства камеры (по умолчанию: 0)
  -v, --verbose        Подробное логирование
```

Сглаживание Калмана, детекция моргания и взвешенная гребневая регрессия включены по умолчанию.

## Горячие клавиши (в режиме отслеживания)

| Клавиша | Действие |
|---------|----------|
| `T` | Включить/выключить режим тренировки |
| `R` | Перекалибровка |
| `V` | Показать/скрыть превью видео |
| `Esc` | Выход |

## Разработка

### Первоначальная настройка

```bash
make setup
```

Команда автоматически:
1. Устанавливает Python 3.12 (`brew install python@3.12` на macOS, `winget install` на Windows)
2. Настраивает Poetry-окружение на Python 3.12 (`poetry env use`)
3. Устанавливает все зависимости (`poetry lock && poetry install`)

Требования: [Homebrew](https://brew.sh) на macOS или [winget](https://learn.microsoft.com/windows/package-manager/winget/) на Windows, а также [Poetry](https://python-poetry.org/docs/#installation).

### Установка зависимостей (если Python уже настроен)

```bash
make install
```

### Тестирование

```bash
make test           # или poetry run pytest
make test-verbose   # или poetry run pytest -v
```

30 тестов покрывают: утилиты, регрессию, детекцию моргания, измерение точности.

### Все команды Makefile

```bash
make setup          # Установка Python 3.12 + Poetry env + зависимости
make install        # Установка зависимостей (Python уже настроен)
make run            # Запуск из исходников
make test           # Запуск тестов
make test-verbose   # Запуск тестов с подробным выводом
make build          # Сборка нативного приложения
make clean          # Очистка __pycache__, build, dist
make help           # Справка
```
